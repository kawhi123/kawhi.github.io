发表于 2019-02-16 | 分类于 writeup ， BugkuCTF

thinking

  1. error_reporting(0) : 关闭所有错误报告。

  2. include （或 require）语句会获取指定文件中存在的所有文本/代码/标记，并复制到使用 include 语句的文件中。

  3. isset — 检测变量是否已设置并且非 NULL。

  4. preg_match : 执行匹配正则表达式。
    /*****/斜杠内表示的是正则表达式，^表示以该字符串为开头，$表示以该字符串为结尾，\w一般都是指包含大小写字母数字和下划线 相当于([0-9a-zA-Z])，+表示该字符串出现一次或多次。

  5. var_dump() 函数显示关于一个或多个表达式的结构信息，包括表达式的类型与值。数组将递归展开值，通过缩进显示其结构。

  6. eval() 函数把字符串按照 PHP 代码来计算。该字符串必须是合法的 PHP 代码，且必须以分号结尾。如果没有在代码字符串中调用 return 语句，则返回 NULL。如果代码中存在解析错误，则 eval() 函数返回 false。

  7. $和$$
    $args(一个美元)是一个普通变量，名称为：args，存储任何值，如：string，integer，float等。
    $$var(两个美元)是一个可变变量，一个可变变量获取了一个普通变量的值作为这个可变变量的变量名。

  8. $GLOBALS — 引用全局作用域中可用的全部变量。
    $GLOBALS是一个超全局变量，包含了全部变量的全局组合数组。在$GLOBALS 数组中，每一个变量为一个元素，键名对应变量名，值对应变量的内容。

你必须让他停下

发表于 2019-01-28 | 更新于 2019-01-30 | 分类于 writeup ， BugkuCTF

ctf

地址：http://123.206.87.240:8002/web12/

thinking

打开链接发现页面不停地刷新，查看源代码可以发现内嵌了一个<script>标签。

setTimeout()用于设置一个定时器，reload()方法用于刷新当前页面。因此通过浏览器禁用JavaScript可以得到flag。

源代码最后有一句flag is here，通过抓包应该也能找到flag。

tip

方法一：禁用JavaScript

  1. 打开Google首页，点击右上角的自定义及控制Google Chrome按钮，进入设置>高级>内容设置>Javascript，将其屏蔽。

  2. 刷新题目的网页，当图片加载出来后查看网页源代码（Ctrl+U）,就可以拿到flag：flag{dummy_game_1s_s0_popular}。

方法二：Burp Suite抓包

  1. 安装JDK并配置java环境

    网上教程很多，但是大多需要手动配置环境变量，我找到了一个比较简单的方法可以跳过这一步骤。

    • https://www.java.com/en/download/win10.jsp
      进入链接点击Agree and Start Free Download下载应用程序JavaSetup8u201。

    • 点击该应用程序按照提示进行安装（可以更改安装路径），安装结束后就OK了，不需要再配置环境变量。

    • 进入cmd窗口输入指令java -version或者可验证是否安装成功。

  2. 下载Burp Suite 1.7.31破解版并安装

    官网有付费正版，土豪随意。下面是破解版的安装步骤。

    • https://www.52pojie.cn/thread-691448-1-1.html
      这个帖子简直太赞了，附有百度云盘下载链接和提取码，而且有多个版本的burp。

    • 安装有些复杂。双击burp-loader-keygen文件会弹出一个破解界面，将License Text改为www.52pojie.cn,复制下面的License。

    • 点击Run，这时会弹出一个 Enter license key 的安装界面，粘贴刚复制的License。

    • 点击Next>Manual activation>Copy request。粘贴到破解界面中的Activation Request，这时会得到Activation Response，将其粘贴到安装界面中的Paste response，最后点击Next就破解并安装完成了。

  3. 使用burp进行抓包

    https://blog.csdn.net/gitchat/article/details/79168613

    https://www.cnblogs.com/nieliangcai/p/6689915.html

get

  2. reload()方法：https://www.w3cschool.cn/jsref/met-loc-reload.html

  3. Burp Suite：https://resources.infosecinstitute.com/burp-suite-walkthrough/
