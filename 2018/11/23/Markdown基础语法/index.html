Markdown基础语法

标题

Markdown支持两种标题的语法，类Setext和类Atx形式。

  1. 类Setext形式是用=等号和-减号来表示(一个及以上），例如：
    这是一级标题
    ========
    这是二级标题
    ————–
  2. 类Atx形式是在行首插入1-6个#，对应一到六级标题，例如：
    # 这是一级标题
    ## 这是二级标题
    ###### 这是六级标题

注意：#后有一个空格！！！

段落和换行

  1. 一个Markdown段落由一到多个连续的文本行组成，段落前后要有一个以上的空行（若某一行只包含空格和制表符也被视为空行）。

  2. 换行：在一个文本行末键入两个及以上的空格或制表符（Tab键）后回车（Enter键）即可换行。另外，由于Markdown兼容部分HTML语法，我习惯用<br/>来换行。

文本样式

  1. 加粗：在需要加粗的文字前后各加两个*星号或者_底线符号，前后符号要匹配且符号和字之间不能有空格。例如：

    **hello world** 效果是 hello world

  2. 斜体：在需要倾斜的文字前后各加一个*星号或者_底线符号。例如：

    *Markdown* 效果是 Markdown

  3. 倾斜加粗：前后各加三个*星号或者_底线符号。例如：

    ***kawhi*** 效果是 kawhi

  4. 高亮：前后各加一个及以上的“`”反引号（Esc键下方）。例如：
    `#include<stdio.h>` 效果是 #include<stdio.h>

  5. 删除线：前后各加两个~波浪号（Esc键下方）。例如：

    ~~github~~ 效果是 github

  6. 分割线：在单独的一个文本行内用三个及以上的*星号、_底线符号来建立分割线（符号间可以有空格）。例如：

    * * *

    _ _ _
    效果都是

注意：一行内不得出现其他文字和符号！

列表

  1. 无序列表：每个文本行前使用*星号、+加号、-减号来标记。例如：
    + red
    + blue
    + yellow
    效果是

    • red
    • blue
    • yellow

    注意：“+”加号后有一个空格！！！

  2. 有序列表：使用阿拉伯数字接着一个英文句点。例如
    1. red
    2. blue
    3. yellow
    效果是

    1. red
    2. blue
    3. yellow

    注意：英文句点后有一个空格！！！

表格

  1. 每个表头或表格单元之间用“|”来隔开。表头下要有-|-|-，有几个表头就有几个“-”减号。
  2. 左对齐：默认左对齐，或在“-”左侧加“:”表示左对齐。例如：
    -|-|-
  3. 右对齐：在“-”右侧加“:”表示右对齐。例如：
    -:|-:|-:

  4. 居中对齐：在“-”两侧都加“:”表示居中对齐。例如：
    \:-:|:-:|:-:|

    |姓名|班级|学号|
    -|-|-
    张三|1|10001
    李四|2|10002
    王五|3|10003

    效果是

姓名 班级 学号
张三 1 10001
李四 2 10002
王五 3 10003

注意：表格前一定要有空行！！！

区块引用

  1. 基本使用：在需要引用的文本前加上>大于号，若要跳出引用空行即可。例如：
    >Markdown是一种可以使用普通文本编辑器编写的标记语言。
    效果是：

    Markdown是一种可以使用普通文本编辑器编写的标记语言。

  2. 嵌套使用：根据引用文本的层次加上不同数量的>。例如：
    >引用一
    >>引用二
    >>>引用三
    效果是：

    引用一

    引用二

    引用三

    补充：引用的区块内也可以使用其他的 Markdown 语法，包括标题、列表、代码区块等。

代码区块

  1. 每行代码前缩进四个空格或一个制表符即可建立代码区块。
  2. 在需要插入的代码区块的前一行及后一行各使用三个“`”反引号。例如：
    ```
    #include<stdio.h>
    void main()
    {
    printf(“hello world”);
    }
    ```
    效果是：

注意：一个代码区块会一直持续到没有缩进的那一行（或是文件结尾）。
