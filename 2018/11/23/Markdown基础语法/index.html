Markdown基础语法

标题

Markdown支持两种标题的语法，类Setext和类Atx形式。

  1. 类Setext形式是用=等号和-减号来表示(一个及以上），例如：
    这是一级标题
    ========
    这是二级标题
    ————–
  2. 类Atx形式是在行首插入1-6个#，对应一到六级标题，例如：
    # 这是一级标题
    ## 这是二级标题
    ###### 这是六级标题

注意：#后有一个空格！！！

段落和换行

  1. 一个Markdown段落由一到多个连续的文本行组成，段落前后要有一个以上的空行（若某一行只包含空格和制表符也被视为空行）。

  2. 换行：在一个文本行末键入两个及以上的空格或制表符（Tab键）后回车（Enter键）即可换行。另外，由于Markdown兼容部分HTML语法，我习惯用<br/>来换行。

    文本样式

  1. 加粗：在需要加粗的文字前后各加两个*星号或者_底线符号，前后符号要匹配且符号和字之间不能有空格。例如：

    **hello world** 效果是 hello world

  2. 斜体：在需要倾斜的文字前后各加一个*星号或者_底线符号。例如：

    *Markdown* 效果是 Markdown

  3. 倾斜加粗：前后各加三个*星号或者_底线符号。例如：

    ***kawhi*** 效果是 kawhi

  4. 高亮：前后各加一个及以上的

#include<stdio.h>
void main()
{
printf(“hello world”);
}
`
注意：一个代码区块会一直持续到没有缩进的那一行（或是文件结尾）。
