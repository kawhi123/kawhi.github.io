配置

  1. VMware Tools安装及共享文件夹的设置
    https://www.cnblogs.com/huangjianxin/p/6343881.html 

    在虚拟机ubuntu中访问主机共享文件夹的方法:
在终端命令行输入  cd /mnt/ + 按下两次Tab键 ，
回车即可进入主机中的共享文件夹。

  2. 更换源
    https://baijiahao.baidu.com/s?id=1594743811772806499&wfr=spider&for=pc

  3. 安装中文输入法
    下载搜狗输入法linux版：http://pinyin.sogou.com/linux/?r=pinyin
    选择 64 位版本
    注意：完成后使用Ctrl+Space或Shift切换中英文输入法。 

    输入法需要用到 fcitx，所以需要先安装   
apt-get install fcitx fcitx-config-gtk2   
进入到“下载”目录中（可以使用复制粘贴），执行   
dpkg -i sogoupinyin_2.2.0.0108_amd64.deb
